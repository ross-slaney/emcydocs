## Idioma por defecto oculto

Cuando `hideDefaultLocaleInUrl` esta activado, el idioma principal vive en `/docs/...` en vez de `/en/docs/...`.

## Prefijos para otros idiomas

Los idiomas adicionales pueden usar rutas explicitas como `/es/docs/...`.

## Por que importa

Esto deja la ruta principal mas limpia y conserva una estructura clara para las traducciones.

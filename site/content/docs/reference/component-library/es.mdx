## Acordeones

<Accordion>
  <AccordionItem title="Que deberia ir en un acordeon?" defaultOpen>
    FAQ, ramas opcionales de setup, notas de migracion o detalles que no deben romper el flujo principal.
  </AccordionItem>
  <AccordionItem title="Sigue funcionando con markdown normal?">
    Si. Puedes mezclar codigo fenced, headings, listas, tablas y JSX dentro del mismo archivo.
  </AccordionItem>
  <AccordionItem title="Por que no otro paquete UI?">
    Porque EmcyDocs quiere seguir siendo una sola libreria. Estos componentes son pequenos, enfocados en docs y comparten el mismo stylesheet.
  </AccordionItem>
</Accordion>

      </Step>
      <Step title="Usa los layouts compartidos">

      </Step>
      <Step title="Usa componentes dentro del MDX">

      </Step>
    </Steps>
  </TabsContent>

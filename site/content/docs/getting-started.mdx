## Install

```bash
npm install @emcy/docs
```

## Point the package at your docs

Create a docs source in your app:

```ts
import path from "node:path";
import { createDocsSource } from "@emcy/docs";

export const docsSource = createDocsSource({
  contentDir: path.join(process.cwd(), "content/docs"),
  basePath: "/docs",
  defaultLocale: "en",
  locales: ["en", "es"],
  hideDefaultLocaleInUrl: true,
  sectionLabels: {
    "": "Getting Started",
    guides: "Guides",
    reference: "Reference",
  },
  sectionOrder: ["", "guides", "reference"],
});
```

## Render the layout

Use one of the package layouts in your App Router tree:

```tsx
import { DocsLayout } from "@emcy/docs";

export default function Layout({ children }: { children: React.ReactNode }) {
  return (
    <DocsLayout navigation={docsSource.getNavigation()}>
      {children}
    </DocsLayout>
  );
}
```

## Render the page

Resolve the current route from the same source and hand the result to `DocsPage` or `DocsHomePage`.

```tsx
const resolved = docsSource.resolveRoute(slug);
```

That gives you:

- the current entry
- previous and next entries
- section landing redirects
- metadata and static params helpers

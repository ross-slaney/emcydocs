## Best for

Notebook works well when your docs feel closer to a product workspace than a marketing site.

## Character

Compared with the classic docs layout, notebook keeps the same content source and navigation structure but presents them in a tighter shell.

## Route

This example site uses the notebook layout under `/notebook`.

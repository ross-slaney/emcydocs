# 文档与博客共用一套 MDX 管线

当博客正式进入库以后，再维护第二套 MDX 渲染器就没有意义了。

## 共享渲染让创作保持一致

作者可以继续在文档和博客中使用相同的 callout、card、tab 和代码块。

<CardGrid columns={2}>
  <Card title="同一套语法" href="#同一套语法">
    不需要学习两套接近却不同的组件约定。
  </Card>
  <Card title="同一套主题" href="#同一套主题">
    默认就能共享间距、边框和 prose 风格。
  </Card>
</CardGrid>

## 同一套语法

复用同一个组件映射，意味着库只需要维护一份 MDX 契约。

## 同一套主题

文档主题变量已经定义了表面、边框、前景色和密度。博客应该直接消费这些 token。

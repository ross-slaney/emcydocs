## 作用

`createDocsSource(...)` 是这个包的主要入口。

## 它会返回什么

返回的对象会暴露这些辅助函数：

- `getNavigation()`
- `getEntry()`
- `getHomeEntry()`
- `resolveRoute()`
- `getMetadata()`
- `getStaticParams()`
- `search()`

## 为什么它很核心

EmcyDocs 没有把内容加载、路由生成、TOC 提取和搜索拆成一堆分散工具，而是统一放在一个 source 对象后面。

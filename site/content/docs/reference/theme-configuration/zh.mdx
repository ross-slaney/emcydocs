## Theme prop

```tsx
<DocsLayout
  navigation={docsSource.getNavigation()}
  theme={{
    preset: "neutral",
    mode: "light",
    accentHue: 38,
    layoutWidth: "1500px",
    contentWidth: "48rem",
    sidebarWidth: "276px",
    tocWidth: "240px",
    radius: "xl",
  }}
  themeSwitcher={<ThemeSwitcher />}
/>
```

## 参数说明

<Accordion>
  <AccordionItem title="preset 和 mode" defaultOpen>
    `preset` 负责基础色板与表面风格，`mode` 用来在支持的 preset 中强制 light 或 dark。
  </AccordionItem>
  <AccordionItem title="accentHue">
    当你想保留同一 preset，只替换品牌强调色时使用。
  </AccordionItem>
  <AccordionItem title="layoutWidth, contentWidth, sidebarWidth, tocWidth">
    用同一个布局组件覆盖 classic、notebook、embedded 与 minimal 四种体验。
  </AccordionItem>
  <AccordionItem title="radius">
    通过 `md`、`lg` 或 `xl` 调整整体表面的锐利或圆润程度。
  </AccordionItem>
</Accordion>

## 可选 switcher 插槽

<Callout type="tip" title="外壳只提供插槽">
  EmcyDocs 不强制语言切换器或主题切换器的实现。需要时传入 `languageSwitcher` 与 `themeSwitcher`，不需要时直接省略即可。
</Callout>

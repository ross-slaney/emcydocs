EmcyDocs es una libreria de documentacion para equipos de Next.js que quieren MDX dentro del repo, rutas prerenderizadas amigables para SEO, una UX mobile integrada y un solo shell de docs que puede cambiar mediante temas y chrome personalizado.

## Ideas principales

- Una sola llamada a `createDocsSource(...)` construye el modelo de contenido, la navegacion, la metadata, los helpers de rutas, la extraccion de headings y la busqueda.
- Un unico `DocsLayout` controla el shell mientras el sistema de temas maneja presets, densidad, tamanos, acento y overrides opcionales de tokens.
- Cuando el header o el sidebar por defecto no bastan, puedes sustituirlos por tu propio componente o por `null`.
- Carpetas por idioma como `entry/en.mdx`, `entry/es.mdx` y `entry/zh.mdx` se convierten directamente en URLs localizadas.

## Lo que muestra este sitio de ejemplo

- Una sola ruta `docs` con un unico layout
- Un motor de temas headless con presets rapidos y un theme studio avanzado
- Rutas localizadas para ingles, espanol y chino
- Una libreria de componentes para homes, guias y referencias

## Why it matters

Mobile docs are easy to underestimate. Search, navigation, sticky controls, and scroll locking all get awkward quickly if they are rebuilt ad hoc on every project.

## What EmcyDocs ships

- a mobile docs bar
- a mobile nav drawer
- a mobile search entrypoint
- body scroll locking while the drawer is open
- one layout shell that keeps the mobile UX consistent across theme presets

## Outcome

The default package UX covers the annoying parts so product teams can focus on content and branding instead of rebuilding mobile docs chrome from scratch.

## 为什么重要

移动端文档很容易被低估。如果每个项目都临时重做搜索、导航、吸顶控件和滚动锁定，这些体验很快就会变得别扭。

## EmcyDocs 提供了什么

- 移动端文档栏
- 移动端导航抽屉
- 移动端搜索入口
- 抽屉打开时锁定 `body` 滚动
- 同一套布局壳层在不同主题 preset 下保持一致的移动端体验

## 效果

默认体验已经覆盖了最麻烦的部分，因此产品团队可以把注意力放在内容和品牌上，而不是反复重建移动端文档框架。

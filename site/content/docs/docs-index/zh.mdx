EmcyDocs 是一个面向 Next.js 团队的文档库，适合把 MDX 放在仓库内、使用对 SEO 友好的预渲染路由、内置移动端文档体验，并且在独立文档站和嵌入式文档之间自由切换。

## 核心理念

- 一次 `createDocsSource(...)` 调用即可生成内容模型、导航、元数据、路由辅助函数、标题提取和搜索。
- 同一个包既可以驱动独立文档站，也可以驱动嵌入在更大应用中的文档。
- 布局可以互换。示例站点基于同一套内容树展示了 classic、notebook 和 minimal 三种视图。
- 像 `entry/en.mdx`、`entry/es.mdx`、`entry/zh.mdx` 这样的语言文件夹会直接映射到带语言感知的 URL。

## 这个示例站点展示了什么

- 经典的 `docs` 体验
- 更紧凑的 `notebook` 布局
- 更专注阅读的 `minimal` 布局
- 在宿主外壳中渲染文档的 `embedded/docs` 模式

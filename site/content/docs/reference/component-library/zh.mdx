## Accordions

<Accordion>
  <AccordionItem title="什么内容适合放进 accordion？" defaultOpen>
    FAQ、可选安装分支、迁移注意事项，或者不想打断主叙事但又必须说明的补充内容。
  </AccordionItem>
  <AccordionItem title="还能和普通 Markdown 一起使用吗？">
    可以。你仍然可以在同一个 `.mdx` 文件里混合 fenced code、标题、列表、表格和 JSX 组件。
  </AccordionItem>
  <AccordionItem title="为什么不再拆一个 UI 包？">
    因为 EmcyDocs 要保持单一库的边界。这些组件体积小、面向文档、并且由同一份样式系统驱动。
  </AccordionItem>
</Accordion>

## Formas soportadas

EmcyDocs soporta:

- archivos planos como `getting-started.mdx`
- carpetas por idioma como `locale-folders/en.mdx`

## Orden

Usa el frontmatter `order` para controlar el orden dentro de una seccion.

## Secciones

El primer segmento de la ruta normalmente se convierte en la clave de seccion. Eso mantiene la navegacion guiada principalmente por el sistema de archivos, mientras sigue permitiendo configurar etiquetas y orden en la fuente.

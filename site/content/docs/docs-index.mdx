EmcyDocs is a docs library for Next.js teams that want repo-local MDX, SEO-friendly prerendered routes, built-in mobile docs UX, and a clean way to switch between standalone and embedded docs experiences.

## Core ideas

- A single `createDocsSource(...)` call builds your content model, navigation, metadata, route helpers, heading extraction, and search.
- The same package can power a standalone docs site or docs embedded inside a larger app.
- Layouts are interchangeable. The example site exposes classic docs, notebook, and minimal views from the same content tree.
- Locale folders like `entry/en.mdx` and `entry/es.mdx` map directly into locale-aware URLs.

## What this example site shows

- A classic `/docs` experience
- A denser `/notebook` layout
- A stripped-down `/minimal` layout
- An `/embedded/docs` mode that renders docs inside a host shell

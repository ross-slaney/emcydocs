## 隐藏默认语言

启用 `hideDefaultLocaleInUrl` 后，默认语言会出现在 `docs/...`，而不是 `en/docs/...`。

## 次级语言前缀

其他语言仍然可以映射到显式前缀，例如 `es/docs/...` 或 `zh/docs/...`。

## 为什么这很有用

默认语言可以保持最简洁的营销和 SEO 路径，而次级语言依然拥有清晰、可预测的结构。

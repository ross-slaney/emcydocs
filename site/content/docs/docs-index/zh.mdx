## 核心理念

- 一次 `createDocsSource(...)` 调用即可生成内容模型、导航、元数据、路由辅助函数、标题提取和搜索。
- 一个 `DocsLayout` 负责文档外壳，主题系统负责 preset、密度、尺寸与强调色。
- 当默认 header 或 sidebar 不够用时，你可以用自己的组件或 `null` 来替换它们。
- 像 `entry/en.mdx`、`entry/es.mdx`、`entry/zh.mdx` 这样的语言文件夹会直接映射到带语言感知的 URL。

## 这个示例站点展示了什么

- 一个由单一布局组件驱动的 `docs` 路由
- 通过主题工作室调整 preset、密度、圆角和壳层尺寸
- 英文、西班牙文与中文的本地化路由
- 适用于首页、指南和参考页的组件库

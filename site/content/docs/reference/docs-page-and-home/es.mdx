## DocsPage

`DocsPage` renderiza una sola entrada con enlaces de headings, soporte de TOC, navegacion anterior y siguiente, y una accion para copiar el enlace.

## DocsHomePage

`DocsHomePage` renderiza la entrada de inicio junto con tarjetas de seccion derivadas del arbol de navegacion.

## Por que existen ambas

Los sitios de docs normalmente necesitan una experiencia de aterrizaje distinta para la ruta superior. Mantenerla separada hace mas facil darle a la home una vista general amplia sin tratarla como un articulo normal.
